Course Experience





COMP 411
Computer Organization

This course included fundamental concepts such as binary numbers, binary arithmetic, and representing information as well as instructions. I learned to program in MIPS assembly language. It covered the fundamentals of computer hardware design, transistors and logic gates, progressing through basic combinational and sequential components, culminating in conceptual design CPU.



COMP 421
Files and Databases

This course focused on the placement of data on secondary storage, and file organization. In addition we learned about database history, practice, major models, system structure and design.



COMP 410
Data Structures and Algorithms

This course taught me how to organize the data used in computer programs so that manipulation of that data can be done efficiently on large problems and large data instances. Rather than learning to use the data structures found in the libraries of programming languages, I learned how those libraries were constructed, and why the items that were included in them were there. This course involved coursework in Java.



COMP 455
Models of Language and Computation

This course introduced me to the theory of computation with a focus on finite automata, regular languages, pushdown automata, context-free languages, and Turing machines.



COMP 126
Practical Web Design and Development

A ground-up introduction to current principles, standards, and best practice in website design, usability, accessibility, development, and management through project-based skills development in HTML5, CSS, and basic JavaScript.



COMP 283
Discrete Structures

This course introduces discrete structures (sets, tuples, relations, functions, graphs, trees) and the formal mathematics (logic, proof, induction) used to establish their properties and those of algorithms that work with them. Develops problem-solving skills through puzzles and applications central to computer science.
